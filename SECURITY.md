# Security Policy

## Supported Versions

All versions of this project are currently supported with security updates and bug fixes.

| Version | Supported |
| :--- | :--- |
| All Versions | :white_check_mark: |

## Reporting a Vulnerability

To report a security vulnerability or a critical bug, please open a new issue directly in the **Issues** tab.

All reports will be tracked publicly through the Issues section and will be addressed as soon as possible. Thank you for your contributions!
